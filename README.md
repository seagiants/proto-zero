# Proto Zero

## Dev

To get the code:

```bash
git clone https://github.com/seagiants/proto-zero.git
```

To run:

```bash
yarn install
yarn start
```

Of course, you need [Yarn](https://yarnpkg.com) and [Node](https://nodejs.org/).
